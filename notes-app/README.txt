This is basic notes-app built using node.js. This app provides the following feature using corresponding command 
1. Add a note
2. Remove the note
3. List notes
4. Read notes

There are four command added to the app which are 
add - to add note
remove - to remove note
list - to list all notes
read - to read note

Example add command :
node app.js add --title="To Study" --body="node.js"
